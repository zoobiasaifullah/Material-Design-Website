Design Guidelines

Learn the foundational principles and best practices for creating great Material Design experiences

color_lens

Color System

Material Design 3's dynamic color system provides adaptive color palettes derived from user preferences, creating personalized and accessible interfaces.

  • Primary, secondary, and tertiary color roles
  • Automatic light and dark theme generation
  • WCAG accessibility compliance
  • Dynamic color extraction from wallpapers
text_fields

Typography

The type system uses a scale of font sizes and weights to create clear hierarchies and ensure readability across all devices and contexts.

  • Display, headline, title, body, and label scales
  • Google Sans and Roboto font families
  • Line height and letter spacing guidelines
  • Responsive typography best practices
layers

Elevation

Surfaces at different elevations create depth through shadows and tonal changes, establishing spatial relationships and visual hierarchy.

  • Five elevation levels (0-5)
  • Shadow and tonal elevation systems
  • Light and dark mode variations
  • Responsive elevation patterns
motion_photos_on

Motion

Thoughtful motion design focuses attention, expresses brand personality, and provides continuity throughout the user experience.

  • Standard easing curves for natural motion
  • Duration tokens for consistent timing
  • Choreographed transitions
  • Reduced motion accessibility
